How to Install Stream Player on Firestick

Step 1 - Enable Developer Mode

  1. Open Settings
  2. Go to My Fire TV
  3. Select About
  4. Click Fire TV Stick 7 times
  5. Enable Developer Options

Step 2 - Install Downloader

Search and install the Downloader app.

Step 3 - Install the APK

Open Downloader and enter:

yourdomain.com/app/stream-player.apk

Install the application.
