Comment installer Stream Player sur Android TV

Guide complet pour installer le lecteur IPTV sur les téléviseurs Android.

Étape 1 – Activer les sources inconnues

Pour installer des applications externes sur des appareils Android, vous devez d'abord activer l'installation depuis des sources inconnues.

  1. Ouvrez Paramètres sur votre Android TV
  2. Allez dans Sécurité et restrictions
  3. Sélectionnez Sources inconnues
  4. Activez l'option pour le gestionnaire de fichiers ou le navigateur que vous utiliserez

Étape 2 – Installer une application pour télécharger des APK

Vous aurez besoin d'une application permettant de télécharger des fichiers APK depuis Internet.

L'une des plus utilisées est Downloader.

  1. Ouvrez le Google Play Store sur votre Android TV
  2. Cherchez Downloader
  3. Installez l'application

Étape 3 – Télécharger Stream Player

Une fois l'application Downloader installée :

  1. Ouvrez l'application Downloader
  2. Entrez l'URL suivante :

https://tudominio.com/app/stream-player.apk

Puis appuyez sur Go pour commencer le téléchargement.

Étape 4 – Installer l'application

Une fois le fichier APK téléchargé :

  1. L'installateur Android apparaîtra
  2. Appuyez sur Installer
  3. Attendez que l'installation se termine

Ensuite, vous pourrez ouvrir l'application depuis le menu des applications de votre Android TV.

Étape 5 – Configurer Stream Player

Lors de la première ouverture de l'application, vous pourrez entrer vos données IPTV.

Stream Player est compatible avec :

  • Listes M3U
  • Connexion Xtream Codes
  • Bibliothèques de films et séries

Appareils compatibles

Stream Player fonctionne sur la plupart des appareils Android TV :

  • Smart TV avec Android TV
  • Box Android TV
  • Appareils avec Android TV intégré
  • Appareils type Amazon Fire TV Stick compatibles avec l'installation manuelle
