Comment installer Stream Player sur Firestick

Étape 1 - Activer le mode développeur

  1. Allez dans Paramètres
  2. Sélectionnez "My Fire TV"
  3. Entrez dans "À propos"
  4. Appuyez 7 fois sur "Fire TV Stick"
  5. Les options développeur apparaîtront

Étape 2 - Autoriser les applications inconnues

Dans les options développeur activez :

  • ADB Debugging
  • Apps from Unknown Sources

Étape 3 - Installer Downloader

Recherchez l'application Downloader dans la boutique Amazon et installez-la.

Étape 4 - Télécharger Stream Player

Ouvrez Downloader et entrez l'URL suivante :

https://tudominio.com/app/stream-player.apk

Étape 5 - Installer l'application

Une fois le fichier APK téléchargé, appuyez sur Installer.

Étape 6 - Ouvrir Stream Player

Entrez vos identifiants IPTV ou une liste M3U pour commencer.
