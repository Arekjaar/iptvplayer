How to Install Stream Player on Firestick

Step 1 - Enable Developer Mode

  1. Go to Settings
  2. Select "My Fire TV"
  3. Enter "About"
  4. Press 7 times on "Fire TV Stick"
  5. Developer Options will appear

Step 2 - Allow Unknown Apps

In Developer Options enable:

  • ADB Debugging
  • Apps from Unknown Sources

Step 3 - Install Downloader

Search for the Downloader app in the Amazon store and install it.

Step 4 - Download Stream Player

Open Downloader and enter the following URL:

https://tudominio.com/app/stream-player.apk

Step 5 - Install the Application

Once the APK file is downloaded, press Install.

Step 6 - Open Stream Player

Enter your IPTV credentials or an M3U playlist to begin.
